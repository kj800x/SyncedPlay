Install Instructions:

This program doesn't install quite like other programs. This program depends on a few programs to run, so you install those first.

Installing Python

Make sure you install the 2.7 version.

Windows Users:

You get it easier than the rest of them. You just need to go to python.org and install Python. Boom! Python is done! That was easy!

One thing to note, keep track of where it installs it, because you might need that later.

Mac Users:

(Un)lucky, you guys have a little work to do. Apple tried to help you and already installed Python for you. Yay! ... But, they messed it up and it doesn't work with a bunch of things. Not Yay!

You have to go and download a fresh version straight from python.org and install it anew. When you are installing it, pay attention to where you are installing it to. Make a note of that, because you WILL need it later!

Linux Users:

You guys chose the hard life, and I'm going to leave it up to you to figure out how to install python and py-game.

I lied, your instructions are so easy: For Debian, Mint, Ubuntu or similars, try running "apt-get install python python-pygame". Then just run "python code.py" to run the program.

Installing PyGame

All Operating Systems:

Download from pygame.org. It's a really easy install. Make sure you get the download for Python 2.7.

Making the shortcut

Windows

Go to the start menu and type in "notepad".

Write the text: 

python.exe code.py

Click "Save As"

Where it says "Save As Type" chose "All Files". Type in "run.bat" for the filename, and save it in the folder that you downloaded the program to.

Double click the file to start the program

If that doesn't work, delete the file you made, and do the steps over again, this time writing the text: 

PLACEWHEREYOUINSTALLEDPYTHON\python.exe code.py

Replace PLACEWHEREYOUINSTALLEDPYTHON with the actual place you installed python. That should work

Mac

Open up the Application "TextEdit"

Write the text: 

PLACEWHEREYOUINSTALLEDPYTHON/python code.py

Click "Save As"

Save it as "run.sh" in the folder that you downloaded my program to.

Double click the file to start the program.
